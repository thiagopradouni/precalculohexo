Busca Binária

A busca binária é um algoritmo eficiente para encontrar um elemento em um array ordenado. Ele compara o elemento alvo com o elemento do meio do array e, dependendo da comparação, continua a busca na metade esquerda ou direita do array, reduzindo o intervalo de busca pela metade a cada iteração.

Passos do algoritmo:

  1. Inicialização dos limites: Inicializa os limites do intervalo de busca.
  2. Iteração: Enquanto o intervalo de busca não estiver vazio, calcula o elemento do meio, compara-o com o elemento alvo e ajusta os limites do intervalo de busca.
  3. Verificação de elemento encontrado: Retorna o índice do elemento se encontrado, caso contrário, indica que o elemento não está presente no array.

Merge Sort

O Merge Sort é um algoritmo de ordenação eficiente que utiliza a abordagem de dividir para conquistar. Ele divide o array em duas metades, ordena cada metade recursivamente e depois mescla as metades ordenadas para produzir o array final ordenado.

Passos do algoritmo:

  1. Divisão: Divide o array em duas metades.
  2. Ordenação recursiva: Ordena cada metade recursivamente.
  3. Mesclagem: Mescla as metades ordenadas para produzir o array final ordenado.

Insertion Sort

O Insertion Sort é um algoritmo de ordenação simples que percorre o array e insere cada elemento na posição correta em relação aos elementos já ordenados, expandindo assim a porção ordenada do array.

Passos do algoritmo:

  1. Iteração pelo array: Percorre todo o array.
  2. Comparação e inserção: Para cada elemento, compara-o com os elementos à sua esquerda na porção ordenada do array e o insere na posição correta.

Selection Sort

O Selection Sort é um algoritmo de ordenação que divide o array em duas partes: uma parte ordenada e outra desordenada. Em cada iteração, o algoritmo encontra o menor elemento da parte não ordenada e o coloca no início da parte ordenada.

Passos do algoritmo:

  1. Iteração pelo array: Percorre todo o array.
  2. Encontrar o menor elemento: Encontra o índice do menor elemento restante na parte não ordenada.
  3. Troca de elementos: Troca o menor elemento encontrado com o primeiro elemento não ordenado.
  4. Repetição: Repete os passos acima até que todo o array esteja ordenado.

Bubble Sort

O Bubble Sort é um algoritmo de ordenação simples que percorre repetidamente o array, compara elementos adjacentes e os troca se estiverem na ordem errada. Ele continua passando pelo array até que nenhum elemento precise ser trocado, o que indica que o array está ordenado.

Passos do algoritmo:

  1. Iteração pelo array: Percorre todo o array.
  2. Comparação de elementos adjacentes: Compara elementos adjacentes e os troca se estiverem na ordem errada.
  3. Repetição: Repete os passos acima até que nenhum elemento precise ser trocado, indicando que o array está ordenado.
