selection-sort

O Selection Sort é um algoritmo de ordenação que divide o array em duas partes: uma parte ordenada e outra desordenada. Em cada iteração, o algoritmo encontra o menor elemento da parte não ordenada e o coloca no início da parte ordenada.

Passos do algoritmo:

  1. Iteração pelo array: Percorre todo o array.
  2. Encontrar o menor elemento: Encontra o índice do menor elemento restante na parte não ordenada.
  3. Troca de elementos: Troca o menor elemento encontrado com o primeiro elemento não ordenado.
  4. Repetição: Repete os passos acima até que todo o array esteja ordenado.

Bubble Sort

O Bubble Sort é um algoritmo de ordenação básico que percorre repetidamente uma lista, comparando elementos adjacentes e os trocando se estiverem na ordem errada.
